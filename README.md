# Clip_counter_v2
Auto increment after Ctrl+V

ВНИМАНИЕ: для работы нужен CGO.

Запускаешь .exe
В консоли видишь текст.
В буфер обмена помещается 1, а при нажатии Ctrl+V число из буфера обмена вставляется и увеличивается на 1.
Нажмешь в консоли английскую C и Enter, изменишь начальное значение (можно буквы и число на конце)
